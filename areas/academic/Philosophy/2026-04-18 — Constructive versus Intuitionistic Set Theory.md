# Constructive versus Intuitionistic Set Theory

> Stub — to be explored.

## The Core Distinction

Both **constructive** and **intuitionistic** mathematics reject classical logic's law of excluded middle (LEM) and require that mathematical objects be explicitly constructible — but they differ in emphasis and foundations.

| | Constructive | Intuitionistic |
|---|---|---|
| **Origin** | Bishop, Martin-Löf | Brouwer |
| **Rejection** | Non-constructive existence proofs | Classical logic + LEM |
| **Sets** | Sets must have decidable membership | Mental constructions only |
| **Logic** | Intuitionistic logic | Intuitionistic logic |
| **Key feature** | Often compatible with classical math | More radical: math is mental activity |
