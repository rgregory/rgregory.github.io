# Machine Learning Methods Economists Should Know About

**Authors**: Sendhil Mullainathan, Jann Spiess  
**Year**: 2017  
**Citations**: ~1,400+  
**Venue**: *American Economic Review*

## Key Contribution

Bridges machine learning and econometrics, distinguishing prediction (ML strength) from causal inference (econometrics strength). Clarifies when each toolbox applies, how to avoid overfitting in economic contexts, and how to combine both approaches for maximum insight. Technical but accessible pedagogical treatment.

## Intellectual Impact

Opened economics to ML techniques; influenced a generation of applied economists and data scientists. Cross-domain: cited in computer science, public policy, tech industry strategy. Catalyst for broader integration of algorithmic methods into social science.

### Methodological Foundations

- **Underfitting vs. Overfitting** — a fundamental trade-off in learning theory; epistemologically, it asks: how much simplicity do we sacrifice for accuracy?
- **Out-of-Sample Validation** — prediction as the test of understanding; but prediction can succeed without understanding (the Chinese Room problem in epistemology)
- **Causal Inference from Data** — instrumental variables, RCTs, and other econometric tools try to infer causation from observations; ML accepts the limits of pure prediction
- **Goodhart's Law** — when a metric becomes a target, it ceases to be a good metric. ML-driven optimization can make this worse if you're optimizing for the metric rather than the underlying phenomenon
