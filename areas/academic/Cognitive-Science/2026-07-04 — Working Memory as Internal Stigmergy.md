# Working Memory as Internal Stigmergy

## Research Seed

Question: is human **working memory**, as described in Henry Taylor's Scientific American article, functionally analogous to an ant being guided by a chemical directive such as a pheromone trail?

Stronger user hypothesis: working memory is **the same kind of thing** as an ant being led by a chemical directive.

Initial framing: probably **not identical**, but potentially the same *functional pattern* at different scales. Working memory is an internal, limited-capacity, attention-sensitive control buffer. Ant pheromone guidance is an externalized, chemically mediated control cue. Both can be read as directive systems that bias near-term action without requiring a central homunculus or full reflective representation.

## Article Context

The article frames working memory as the information currently available for an active task. It emphasizes:

- working memory is tightly linked to attention;
- information removed from working memory often disappears from consciousness;
- working memory has severe capacity limits;
- global workspace theory treats consciousness as information broadcast broadly across cognitive systems;
- working memory may be spectrum-like rather than simply present/absent.

## Analogy to Ant Chemical Directives

In ant colonies, pheromone trails and other chemical cues are not merely passive signals. They alter action selection: follow this trail, recruit here, avoid there, build here, defend here. The ant does not need a reflective representation of the colony-level goal. A local chemical condition constrains and directs behavior.

Working memory may play a parallel role inside an organism:

- it maintains currently action-relevant information;
- it biases what can guide behavior now;
- it is fragile and easily displaced by new context;
- it can coordinate multiple subsystems without implying a central inner observer;
- it may create the appearance of conscious control because whatever enters it becomes globally usable.

So the analogy is:

```text
pheromone trail : ant action :: working-memory content : organism action
```

Or, more provocatively:

```text
ant follows chemical salience in environment;
human follows attentional salience in working memory.
```

## Key Distinction

The systems differ in substrate and level:

| Feature | Working memory | Ant chemical directive |
|---|---|---|
| Substrate | neural / cognitive | chemical / environmental |
| Location | internal to organism | external/shared environment |
| Scope | one organism's task control | colony/environmental coordination |
| Timescale | seconds to minutes | minutes to days depending on pheromone/context |
| Access | may become conscious | probably not consciously represented by the ant as such |
| Function | maintain and broadcast task-relevant content | bias local behavior and enable distributed coordination |

The better research claim may be:

> Working memory and pheromone-guided ant behavior are both **directive salience systems**: mechanisms that select what matters next and make local action coherent without requiring centralized understanding.

## Research Questions

1. Is working memory best understood as an internal control signal, an internal workspace, or an internal stigmergic substrate?
2. Does consciousness arise when a directive signal is globally broadcast within an organism, whereas ant pheromone signals remain local/environmental?
3. Is the difference between conscious and non-conscious directive systems mainly integration/broadcast depth?
4. Can global workspace theory be compared to stigmergy: many subsystems respond to a shared signal substrate?
5. Does an ant's pheromone-following count as minimal working memory, or does it bypass working memory entirely through stimulus-response coupling?
6. Are there intermediate cases: insects with short-term path integration, reinforcement traces, or working-memory-like neural states?
7. Does this comparison clarify [[Ant Colony Intelligence — Intentional or Functional?]] or merely blur levels of explanation?

## Tentative Thesis

Working memory is not literally the same as an ant pheromone directive, but both may instantiate a common architecture:

> a transient, capacity-limited, action-directing salience state that coordinates behavior without requiring a central self.

In humans, when that directive state is integrated with attention and globally broadcast, we call part of it consciousness. In ants, when a chemical directive coordinates action through the environment, we call it stigmergy. The research problem is whether these are deeply analogous mechanisms or just superficially similar control loops.

## Pending Review Notes

### 2026-07-07 — Directive salience refinement

- Pending durable refinement: working memory and pheromone-guided ant behavior should not be treated as identical mechanisms, but as different implementations of a broader **directive salience** pattern: transient, feedback-sensitive states that bias next action without central command.
- Supporting sources from the first weekly update include global neuronal workspace theory, active-inference accounts of working memory/attention/salience, salience-gated action-selection models, and ant studies where pheromone cues interact with private memory.
- Main caveat: conscious working-memory content appears to involve global neural access/broadcast, while ant pheromone cues are external, shared, and not evidence by themselves of a unified first-person workspace.
