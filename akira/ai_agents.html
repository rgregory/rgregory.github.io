Top findings

1. Claude Code v2.1.205 hardens background/auto-mode operation. It blocks session transcript tampering in auto mode, asks before unresolved dangerous rm -rf variables, improves background task status/resume behavior, and states explicitly when no human input occurred so fabricated transcript approvals are not acted on. 2. OpenAI Codex makes skill-root discovery passive and fail-fast. skills/list now avoids starting executors, waiting for recovery, or reconnecting failed environments when taking a read-only selected-root snapshot. 3. AgentLens evaluates whole coding-agent trajectories. It adds production-assessed trajectory reviews and side-by-side comparisons to catch behavior/product regressions beyond final pass/fail. 4. SkillCenter packages source-grounded skills as SQLite FTS bundles. Its reported 216,938 structured skills use source grounding and offline-searchable SQLite FTS5 bundles, directly relevant to local-first agent memory and skill libraries. 5. Institutional red-teaming treats deployment rules as causal variables. IABench-CA shows multi-agent safety changes materially when only deployment/consequence rules vary.
